Script Management Cheat Sheet

Per-Site Cleanup (one time)

1.Scan site in audit app - flags bad scripts, detects GTM + Cookiebot IDs auto
2.Check GTM container has Cookiebot tag + consent triggers + Consent Mode V2
3.Move flagged tracking scripts into GTM as tags with consent triggers
4.Open Webflow Designer - remove tracking scripts from custom code boxes manual
5.Push GTM + Cookiebot snippets via MCP (add_inline_site_script) auto
6.Remove GTM + Cookiebot from Designer custom code (now API-managed) manual
7.Re-scan to verify clean auto

New Site Setup

1.Create/assign GTM container with Cookiebot + consent triggers
2.Push GTM + Cookiebot snippets via MCP auto
3.All tracking goes into GTM - never into Webflow custom code
4.Scan to verify auto

Where Scripts Go

TypeWhere
Tracking (analytics, pixels, chat widgets)GTM container - as tags with consent triggers
Cookiebot + GTM snippetsWebflow head - via API/MCP
JSON-LD / structured dataWebflow head - via API/MCP
Webflow framework scriptsLeave alone
Rule: Never paste tracking scripts into Webflow's custom code boxes. They go in GTM. Non-tracking scripts go via the API/MCP so we can manage them.
